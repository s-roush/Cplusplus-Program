# Cplusplus-Program
#Summarize the project and what problem it was solving.
This project was about making a grocery tracker that will read items from a text file and find how often that item was purchased. Then the program would allow the user to view a specfic item and how often it was purhcased, view a list of all the items and their frequencies, or view a histogram for all items and their amount of frequencies. This program would help people to analyze what products were selling well, and what products were selling poorly. This would help them order specfic items, help to promote items that aren't selling well, and more.
